Project Overview

Problem Statement

Managing college enquiries manually is time-consuming and inefficient. Staff are overwhelmed with repetitive questions about admission, fees, courses, hostels, timetables, and scholarships.

Feasibility Study

A domain-specific chatbot designed exclusively for the college enquiry system is highly feasible. It reduces manual workload, handles concurrent queries, and provides accurate, instant responses without complex ML models.

Expected Outcome

Automated replies ensuring fast student support and a 24/7 enquiry system. Significantly reduced staff workload and an improved overall experience for prospective and current students.

AI Techniques & Architecture

AI Technique

Rule-based chatbot using pattern matching and predefined robust responses. It evaluates user input against a set of predefined rules to formulate an answer.

Knowledge Representation

Pattern Rules: Simple and effective If-Then condition mapping. e.g. If user asks 'admission' → reply with admission info.

# Rule-Based College Enquiry Chatbot (Pattern Matching)

def get_response(user_input):
    input_text = user_input.lower()
    
    # Pattern matching rules dictionary
    rules = {
        "admission": "Admission is open for 2026. Process: Online Registration -> Entrance Test -> Merit List. Visit the admission portal.",
        "course": "We offer B.Tech, M.Tech, BCA, MCA, BBA, and MBA. For a detailed syllabus, please check the academics tab.",
        "fee": "Fees vary by course: B.Tech is $4000/yr, BCA is $2000/yr. You can pay in 2 installments.",
        "hostel": "Yes, we have separate AC/Non-AC hostels for boys and girls. The fee is $1200 per year including mess.",
        "timing": "College timings are Monday to Saturday, 9:00 AM to 4:30 PM.",
        "locat": "We are centrally located in University City, 5 mins from the main metro station.",
        "document": "You need: 10th & 12th Marksheets, Transfer Certificate, ID Proof, and 4 Passport photos.",
        "scholarship": "Yes! Merit-based scholarships up to 50% waiver are available for students scoring above 90%.",
        "contact": "Contact us at info@edu.college.edu or call 1800-123-4567 for direct assistance."
    }
    
    # Iterate through patterns to find a match
    for keyword, response in rules.items():
        if keyword in input_text:
            return response
            
    # Default fallback response
    return "I'm sorry, I don't have information on that. Please ask about admission, courses, fees, hostels, timing, or contact details."

# Main loop for terminal execution
if __name__ == "__main__":
    print("🤖 College Chatbot started. Type 'exit' to quit.")
    while True:
        user_query = input("You: ")
        if user_query.lower() == 'exit':
            break
        print("Bot:", get_response(user_query))
